# zzosort
Javascript sort module.

## Algorithms used
### Introsort
### Heapsort
### Insertionsort
https://www.geeksforgeeks.org/know-your-sorting-algorithm-set-2-introsort-cs-sorting-weapon/

## Install
```bash
npm install @zzopark/zzosort
```

## Build & Test
```bash
# Transpilling zzosort.js
npm run build
# Testing zzosort.js
npm test
```

## Example
```bash
# Open example.html
npm start
```

## In Browser
```html
<script src="dist/zzosort.js"></script>
```
